# Language Translation Tool

## Overview

This project is a Language Translation Tool developed as part of the CodeAlpha Artificial Intelligence Internship.

The application allows users to translate text between multiple languages using AI-powered translation.

## Features
- Translate text between multiple languages
- Simple and interactive Streamlit interface
- Supports English, Hindi, Telugu, French, Spanish, and German
- Fast and accurate translations
- Text-to-Speech support for translated text
- User-friendly design

## Technologies Used

* Python
* Streamlit
* Deep Translator

## Installation

1. Clone the repository

```bash
git clone <repository_link>
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Run the application

```bash
streamlit run app.py
```
