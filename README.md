- Fast and accurate translations

An NLP-based chatbot that answers user questions using similarity matching techniques.

### Features
- Interactive chatbot interface
- TF-IDF Vectorization
- Cosine Similarity Matching
- Chat History Support
- Intelligent question matching

### Technologies Used
- Python
- Streamlit
- Scikit-learn
- NLP Techniques

## 🎯 Task 4: Object Detection & Tracking

An AI-powered computer vision application that detects and identifies objects in images using YOLOv8.

### Features
- Image Upload Support
- YOLOv8 Object Detection
- Bounding Box Visualization
- Confidence Scores
- Object Counting
- Detection Statistics
- Detection Summary Table
- Streamlit-based User Interface

### Technologies Used
- Python
- Streamlit
- YOLOv8
- OpenCV
- NumPy
- Pillow

# ▶️ How to Run

### Clone Repository

```bash

```

### Navigate to Project Folder

```bash

```

### Install Dependencies

```bash
pip install -r requirements.txt
```

```bash
streamlit run app.py
```
